3. DISTRIBUTION
-------------------------------------------------------------------------------
Catalog of applications, tools, modules and files included in the distribution:

  * Notepad++

    - 8.6.1 32-bit/64-bit

  * Notepad++ PythonScript plugin

    - 3.0.17.0 32-bit/64-bit

      - Python 3.12.1 32-bit/64-bit (original, without Windows 7 support)

  * Notepad++ PythonScript plugin Python updates

    - Python 3.12.1 dlls (patched: https://github.com/adang1345/PythonWin7)

      - x64

        - api-ms-win-core-path-l1-1-0.dll
        - python3.dll
        - python312.dll
        - vcruntime140.dll
        - vcruntime140_1.dll

      - x86

        - api-ms-win-core-path-l1-1-0.dll
        - python3.dll
        - python312.dll
        - vcruntime140.dll

    - Python 3.12.1 modules

      - psutil 5.9.7

  * WinMerge

    - 2.16.36 32-bit/64-bit

See the installation log in autogenerated log directory - `.log`, for details
on executables and setups in the catalog.
